# Public assets

- **logo-design.pdf** – Your Untitled design (logo). Use this as the app logo by exporting it to PNG.
- **logo.png** – Header logo. Export the logo from `logo-design.pdf` (e.g. open the PDF, screenshot or “Export as image” the logo area) and save it here as `logo.png`. The header will then show it; if `logo.png` is missing, the built-in SVG logo is used.
